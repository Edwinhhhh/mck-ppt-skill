# references

这个目录用于存放与 `mck-ppt-skill` 配套的补充资料。

## 建议放入的内容

可以根据需要逐步补充以下类型的文件：

- 行业案例
- 常见 deck 结构模板
- 标题写法示例
- storyline 范式
- chart suggestion 示例
- 管理层汇报风格约束
- 特定行业术语与表达偏好

## 使用建议

- 将长期稳定的补充规范放在这里
- 将高复用的行业模板沉淀在这里
- 避免把临时对话内容直接堆在仓库根目录
- 如果后续扩展多个行业版本，可以按子目录继续拆分

例如：

- `references/retail.md`
- `references/saas.md`
- `references/ops-review.md`

## 目标

这个目录的目标，是让 Skill 不只是一个说明文件，而是逐步演化成一个可维护、可复用、可扩展的咨询式 PPT 大纲生成仓库。
